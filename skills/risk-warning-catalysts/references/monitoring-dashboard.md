# Monitoring Dashboard Guide

Prefer a compact set of high-signal indicators.

## Suggested Categories

- Orders and execution
- Cash flow and receivables
- Leverage and refinancing
- Governance and shareholder behavior
- Policy and procurement
- Utilization, commissioning, or ramp-up

## Good Monitor Design

- One metric per row
- One threshold per metric
- One primary source link
- One follow-up action if the threshold breaks

Example threshold language:

- "Operating cash flow stays negative for two consecutive reporting periods"
- "Short-term debt ratio exceeds 50% without matching refinancing disclosure"
- "Major project revision or cancellation announced"
