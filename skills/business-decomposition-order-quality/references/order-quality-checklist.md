# Order Quality Checklist

Check these items for every major order or project.

| Item | Why It Matters |
| --- | --- |
| Counterparty | Tests customer quality and concentration |
| Signed amount | Distinguishes headline value from actual economics |
| Contract term | Shows whether revenue is one-off or recurring |
| Payment terms | Determines cash-conversion quality |
| Delivery milestone | Tests execution visibility |
| Revenue-recognition basis | Prevents mixing booked revenue with signed orders |
| Change or cancellation clause | Identifies contract fragility |
| External validation | Procurement notice, park website, government release |

If any field is missing, mark it clearly rather than guessing.
