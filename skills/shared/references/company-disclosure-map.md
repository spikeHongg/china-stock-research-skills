# Company Disclosure Map

Use this map to locate the right document before analysis starts.

## Core Company Documents

| Document Type | Main Questions It Answers | Priority Source |
| --- | --- | --- |
| Prospectus | Original business model, fundraising use, risk factors, historical numbers | CNInfo / exchange portal |
| Annual report | Segment mix, strategy, capex, debt, customers, related parties, litigation | CNInfo / exchange portal |
| Interim report | Mid-year trend changes, working capital, order execution, updated risks | CNInfo / exchange portal |
| Quarterly report | Short-cycle revenue, profit, cash flow, balance sheet change | CNInfo / exchange portal |
| Earnings pre-announcement | Profit warning, asset impairment, one-off events | CNInfo / exchange portal |
| Board and shareholder resolutions | Financing, guarantees, acquisitions, major capex, asset disposals | CNInfo / exchange portal |
| Inquiry letters and replies | Accounting quality, unusual contracts, regulators' focus areas | CNInfo / exchange portal |
| Convertible bond or placement filings | Funding need, project economics, leverage plan, dilution | CNInfo / exchange portal |

## Special Topics

| Topic | Typical Documents | Priority Source |
| --- | --- | --- |
| Large order and project pipeline | Contract announcements, bid notices, operator procurement portals | Exchange disclosure + procurement portal |
| Asset impairment and bad debts | Earnings pre-announcement, annual report notes, inquiry replies | CNInfo |
| Equity pledge and insider actions | Pledge announcements, shareholding change filings | CNInfo |
| Lawsuits, guarantees, contingencies | Annual report notes, temporary announcements | CNInfo |
| Capacity and expansion | Fixed-asset filings, local approvals, financing use-of-proceeds | CNInfo + local government portals |

## Retrieval Sequence

1. Pull latest annual, interim, and quarterly reports.
2. Pull all major temporary announcements since the latest annual report.
3. Pull financing and regulator-question documents if the company is in transition.
4. Pull external validation: procurement, policy, statistics, regional approvals.

## Output Reminder

When citing a company fact, always mention which disclosure type it came from.
