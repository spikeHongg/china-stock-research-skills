# Source Registry

Use this registry to prioritize public, accessible, and checkable sources.

## Tier 1: Statutory Disclosures and Regulators

| Source | What to Use It For | URL |
| --- | --- | --- |
| Juchao / CNInfo | Annual reports, interim reports, board resolutions, financing plans, risk disclosures | https://www.cninfo.com.cn/new/index |
| SSE listed company portal | Main-board disclosures for Shanghai-listed issuers | https://www.sse.com.cn/assortment/stock/list/info/company/index.shtml |
| SZSE disclosure portal | Shenzhen-listed issuer disclosures | https://www.szse.cn/disclosure/listed/fixed/index.html |

| National Enterprise Credit Information | Legal entity status, shareholders, administrative filings | https://www.gsxt.gov.cn/index.html |

## Tier 3: Company Channels and Supplementary Public Sources

| Source | What to Use It For | URL |
| --- | --- | --- |
| Company IR site | Presentations, investor Q&A, event notices | Use issuer official website |
| Exchange interactive Q&A | Investor questions and management replies, if publicly archived | Use exchange-linked issuer pages |
| Government and park websites | Local project approvals, land, energy, industrial policy | Use local government official portals |
| Industry media | Secondary context only, never sole support for a core conclusion | Use original article URL |

## Mandatory Rules

- Prefer Tier 1 for all core facts, numbers, contracts, financing, and governance items.
- Use Tier 2 to validate market size, policy, procurement, and regional industry context.
- Use Tier 3 only as supplementary context unless it reproduces an official source.
- Keep every URL directly clickable.
- If a source is behind login or is not publicly reachable, mark it as inaccessible and do not rely on it as the main proof.

## Recommended Source Order by Topic

- Company history and business mix: CNInfo -> annual report -> prospectus -> official website
- Orders and projects: exchange announcement -> operator procurement portal -> local government/park website
- Debt and financing: annual/interim report -> bond/convertible bond filing -> board resolution
- Industry policy: MIIT -> NDRC or local government -> industry association
- Macro and regional demand: National Bureau of Statistics -> province/city statistics bureau -> policy documents
