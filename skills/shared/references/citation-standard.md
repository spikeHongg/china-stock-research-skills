# Citation Standard

Every output must separate `Facts`, `Interpretation`, and `Open Questions`.

## Minimum Citation Unit

- Every key sentence must carry at least one accessible source link.
- Every row in a critical table must carry a primary source.
- Every numeric chart must list the source set below the chart.

## Required Citation Format

Use this format inline:

```text
[来源: 机构或公司, 文档名, 日期, 链接, 页码/章节/公告编号]
```

Example:

```text
2025H1 IDC 业务收入占比达到 100%。[来源: 城地香江, 2025 年半年度报告, 2025-08-XX, https://www.cninfo.com.cn/, 第 X 节]
```

## Evidence Tiers

- `Tier 1`: statutory filings, regulator releases, exchange disclosures
- `Tier 2`: ministry data, public procurement notices, public statistics, official association materials
- `Tier 3`: company websites, media, broker notes, interviews, conference notes

## Judgment Rules

- A core investment conclusion must be supported by at least one Tier 1 or Tier 2 source.
- A Tier 3 source may enrich context but may not be the sole support for a critical conclusion.
- If different sources conflict, present both and state the conflict explicitly.

## Writing Rules

- Tag unverified but plausible reasoning as `Interpretation`, not `Fact`.
- Tag missing evidence as `Open Question` with a clear next source to check.
- Preserve the original unit, period, and accounting scope when quoting numbers.
- Mark whether the figure is audited, reviewed, or unaudited when known.

## Link Rules

- Use direct, accessible URLs whenever possible.
- If a disclosure portal uses a searchable landing page instead of a stable document URL, cite both the portal and enough metadata to re-find the document.
- Never cite a screenshot without the underlying public URL.
