# Research Output Template

Use this structure when merging outputs from multiple skills.

Default all sections to Simplified Chinese. See `skills/shared/references/language-output-standard.md`.

## 1. 核心结论

- 一段式结论摘要
- 主要看多点
- 主要看空点
- 结论置信度

## 2. 事实

- 只写事实
- 每条事实都必须带引用

## 3. 解读

- 解释这些事实意味着什么
- 明确写出假设
- 引用保留在所依赖的事实上

## 4. 反证

- 哪些信息会削弱或推翻当前判断
- 哪些指标或事件会改变观点

## 5. 待验证问题

- 仍需进一步核实的未知点
- 下一步应去核查的具体来源

## 6. 监测指标

- 需要月度或季度跟踪的指标
- 关键触发事件与阈值

## 7. 情景与估值区间

- 悲观 / 中性 / 乐观情景
- 关键假设与敏感性因素
- 有来源支撑的可比对象

## 8. 来源附录

- 完整引用清单
- 链接清单
- 数据限制与证据缺口
