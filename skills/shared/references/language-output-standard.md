# Language Output Standard

Default all working outputs to Simplified Chinese.

## Default Rule

- Unless the user explicitly requests another language, every intermediate and final output should be written in Simplified Chinese.
- This applies to routing decisions, module outputs, scorecards, monitoring dashboards, valuation scenarios, and final merged research frameworks.

## Keep Original Terms When Needed

- Keep source document titles in their original language when citing them.
- Keep technical, regulatory, accounting, or product terms in English when translation would reduce precision.
- When keeping an English term, explain it in Chinese if the meaning may be unclear.

## Section Labels

Use Chinese labels by default, for example:

- `事实`
- `解读`
- `待验证问题`
- `监测指标`
- `情景与估值钩子`
- `来源附录`

## Citation Rule

- Keep URLs unchanged.
- Keep company names, regulator names, and document titles in the original form when needed for source retrieval.
- Surround every key statement with a Chinese sentence structure even if the cited document is in English.

## Override Rule

- If the user explicitly asks for English output, switch the output language to English.
- If the user asks for bilingual output, provide Chinese first and English second unless the user specifies otherwise.
