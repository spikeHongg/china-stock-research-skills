# 路线图 / Roadmap

[中文](#中文) | [English](#english)

## 中文

### 近期计划

- 增加更多可复用 overlays，例如金融、公共事业、国企治理等场景
- 增加示例输出，至少覆盖一只周期股、一只消费股、一只医药股
- 完善 GitHub 远程安装的 smoke test 指引
- 随着版本发布，逐步维护简洁的 `CHANGELOG.md`

### 中期计划

- 在有助于传播和协作的前提下，继续完善中英双语文档
- 增加更多适用于 memo、投委会讨论、跟踪报告的模板
- 完善按行业监管口径组织的 source registry

### 当前暂不纳入范围

- 实时行情或盘口数据接入
- 自动化交易工作流
- 私有终端或专有数据库集成

## English

### Near Term

- Add more reusable overlays for financials, utilities, and state-owned enterprise governance cases
- Add example outputs covering at least one cyclical stock, one consumer stock, and one healthcare stock
- Improve remote GitHub install smoke-test instructions
- Maintain a compact `CHANGELOG.md` as releases begin

### Medium Term

- Continue improving bilingual documentation where it helps adoption and collaboration
- Add more templates for memo-style, committee-style, and follow-up research outputs
- Refine source registries organized around sector-specific regulators and disclosure paths

### Out of Scope for Now

- Real-time market data integration
- Automated trading workflows
- Proprietary terminal or private database integrations
