# Quickstart Prompts

## Start Any New Stock

```text
使用 china-stock-research-orchestrator skill 分析这家 A 股公司，识别主要分析模式，选择需要调用的模块，并返回一份带可访问来源链接的研究框架，结构必须包含：事实、解读、待验证问题。所有中间与最终输出默认使用中文。
```

## Check Whether Growth Quality Is Real

```text
使用 business-decomposition-order-quality skill 判断这家公司披露的增长，是否由真实交付、健康客户质量和实际回款支撑。所有关键结论都要附可访问来源链接，并默认使用中文输出。
```

## Diagnose Financial Stress

```text
使用 financial-health skill 诊断这家公司为什么收入增长没有转化成利润和现金流。请严格区分：事实、解读、待验证问题，并默认使用中文输出。
```

## Build A Monitoring Dashboard

```text
使用 risk-warning-catalysts skill 为这只股票建立月度监测面板，列出关键阈值、来源链接和后续核查动作，默认使用中文输出。
```
